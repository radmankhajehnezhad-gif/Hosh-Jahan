# Dark Room — Public AI Chat (Replit / Render)

این پروژه یک اپ چت ساده با Gradio است که می‌توانید روی Replit یا Render اجرا کنید.

## فایل‌های داخل بسته
- `app_replit.py` — کد برنامه (پیش‌فرض از مدل `google/flan-t5-base` استفاده می‌کند)
- `requirements.txt` — بسته‌های لازم
- `.replit` — تنظیم اجرا در Replit
- `README.md` — همین فایل

## اجرای سریع در Replit
1. یک Repl جدید بساز (Python).
2. تمام فایل‌های این بسته را در Repl آپلود کن.
3. از Shell یا از UI بسته‌ها را نصب کن:
```
pip install -r requirements.txt
```
4. روی Run کلیک کن یا:
```
python app_replit.py
```
5. آدرس نمایش داده‌شده را باز کن — هر کسی با آن لینک می‌تواند چت کند.

## نکات و محدودیت‌ها
- مدل‌های بزرگ ممکن است روی Replit با محدودیت CPU/ریسورس اجرا نشوند. اگر مدل به‌خوبی اجرا نشد:
  - مدل را به یک مدل کوچک‌تر تغییر بده (مثلاً `flan-t5-small` اگر موجود باشد).
  - یا از یک سرویس استریم/API مثل Hugging Face Inference API استفاده کن (نیاز به توکن و گاهی هزینه دارد).
- برای کیفیت بهتر فارسی، بهتر است مدل instruction-tuned فارسی یا دو زبانه پیدا کنی.
- اگر می‌خواهی من نسخه‌ای با پشتیبانی از آپلود فایل، لاگین ساده یا محدودسازی دسترسی درست کنم — بگو تا اضافه کنم.
